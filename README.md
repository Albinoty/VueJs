# Tuto VueJs

Le repo de permet de voir comment fonctionne VueJs via un cours


## Project setup
```BASH
npm install
```

### Compiles and hot-reloads for development
```BASH
npm run serve
```

### Compiles and minifies for production
```BASH
npm run build
```

### Lints and fixes files
```BASH
npm run lint
```
